# 백년양말 폰 발주 웹앱 — 설계서

작성일: 2026-06-26

## 1. 목적 / 문제

사용자(백년양말 도매 발주자)는 매번 주문할 때마다 네이버 카페에 이미지로 올라온
"본사 카탈로그"를 보면서 제품을 하나하나 눈으로 확인하고, 메모장에 필요한 품목을 손으로
적어 카카오톡으로 본사에 보낸다. 매일 반복되는 작업이라 번거롭다.

**목표:** 폰에서 색상·수량만 탭하면 카톡에 바로 붙여넣을 수 있는 발주서 텍스트가
자동으로 만들어지는 도구.

## 2. 사용 시나리오

1. 폰 홈화면 바로가기로 웹앱을 연다.
2. 제품 카드에서 원하는 색상의 수량(죽)을 `－ / ＋` 버튼으로 정한다.
3. 화면 하단에서 총 죽수·예상금액을 참고로 확인한다.
4. `［발주서 만들기］` → 생성된 텍스트를 `［복사］`.
5. 카카오톡에 붙여넣어 본사로 전송.

## 3. 발주서 출력 형식 (확정)

수량이 1죽 이상인 색상마다 한 줄:

```
제품명 / 사이즈 / 색상 / N죽
```

예:
```
고탄력 스타킹덧신 / Free / 블랙 / 3죽
고탄력 미끄럼방지 스타킹덧신 / Free / 커피 / 1죽
```

- 가격·합계는 **복사 텍스트에 포함하지 않는다** (본사 전송용이므로). 화면에만 참고 표시.
- 주문 단위는 **색상별 1죽(10켤레)** 단위. 수량 입력은 죽 단위 정수.

## 4. 데이터 모델

카탈로그는 코드 내장 JSON. 제품 1개 =

```js
{
  name: "고탄력 스타킹덧신",
  size: "Free",              // Free / M / L / "문의"
  colors: ["스킨", "커피", "블랙"],
  price: 250,                // 켤레당 원, 숫자. "문의" 제품은 null
  origin: "수입",            // 대한민국 / 수입
  note: ""                   // 선택. 특가양말 등 특이사항
}
```

### 색상 표시
색상 이름 옆에 색깔 동그라미를 그려 시각적으로 구분한다.
매핑: 스킨=살구톤, 커피=갈색, 블랙=검정. (그 외 색상은 추가 시 매핑 보강)

### 예외: 특가 양말
사이즈·색상·가격이 모두 "문의"인 제품. 색상별 수량 대신 **자유 메모 입력칸** 하나를 둔다.
메모가 비어있지 않으면 발주서에 `특가 양말 / 문의: <메모>` 형태로 한 줄 추가.

## 5. 화면 구성 (단일 HTML 페이지, 모바일 우선)

- 상단: 제목 "백년양말 발주"
- 제품 카드 목록:
  - 제품명, 사이즈 뱃지, (참고)가격
  - 색상별 행: 색깔 동그라미 + 색상명 + `－  0  ＋` (죽)
- 하단 고정 바: `총 N죽 · 예상 ₩XXX,XXX` + `［발주서 만들기］`
- 발주서 모달/영역: 생성된 텍스트(textarea) + `［복사］` + `［초기화］`

## 6. 동작 / 상태

- 수량 상태는 메모리 + `localStorage`에 저장 → 앱 다시 열어도 직전 입력 유지(선택).
- `［초기화］`로 전체 수량 0으로.
- 복사는 `navigator.clipboard` 사용, 실패 시 textarea select 폴백.

## 7. 기술 / 의존성

- **단일 HTML 파일** (`index.html`), 외부 라이브러리 없음(바닐라 JS/CSS).
  - 오프라인·느린 폰에서도 빠르고, 호스팅·갱신이 단순.
- 제품 이미지: v1에서는 미사용(색깔 동그라미로 대체). 추후 필요 시 추가.

## 8. 호스팅 / 갱신

- **GitHub Pages**에 배포 → 고정 URL, 폰 홈화면 바로가기.
- 카탈로그 변경(월 1~2회): 사용자가 새 카탈로그 사진 전달 → Claude가 데이터 갱신 후 재배포(같은 URL 유지).

## 9. 범위 밖 (YAGNI)

- 로그인/사용자 계정, 주문 이력 DB, 자동 카페 크롤링/OCR, 결제, 제품 사진 표시(v1).

## 10. 성공 기준

- 폰에서 카탈로그 전 품목을 탭으로 선택 가능.
- 생성 텍스트가 `제품명 / 사이즈 / 색상 / N죽` 형식과 정확히 일치.
- 복사 버튼으로 카톡에 한 번에 붙여넣기 가능.
